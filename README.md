# Nutrivida

Site institucional desenvolvido para conscientizar sobre segurança alimentar e os impactos ambientais que afetam a produção e o acesso a alimentos.

## Sobre o projeto

O Nutrivida é um site estático de múltiplas páginas, criado como projeto acadêmico, com o objetivo de apresentar de forma clara e visual os problemas ambientais relacionados à alimentação e seus impactos na sociedade.

## Páginas

- Home — Página inicial de apresentação do projeto
- Problemas — Conteúdo sobre os impactos ambientais na produção de alimentos, com gráficos e imagens ilustrativas
- Sobre Nós — Informações sobre a equipe/proposta do projeto
- Contato — Formulário de contato para o usuário enviar nome, e-mail e telefone

## Tecnologias utilizadas

- HTML5 — Estruturação do conteúdo
- CSS3 — Estilização e layout (arquivos de estilo separados por página)

## Objetivo

Projeto desenvolvido para praticar conceitos de front-end como estruturação semântica em HTML, estilização em CSS, criação de menu de navegação (navbar) e formulários.

## Como visualizar

Basta baixar ou clonar o repositório e abrir o arquivo home.html em qualquer navegador.
